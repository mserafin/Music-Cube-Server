# Music-Cube-Server

Server written in Node.js to receive data from the [Music-Cube-Hardware](https://github.com/mserafin/Music-Cube-Hardware) project via the HTTP protocol.



## Setup

```
$ yarn install --pure-lockfile
```

## Start

```
$ yarn start 3000
```
